A library that defines common types and functionality that is needed by various Rust projects. The scope of this library is meant to be limited to core chain definitions. At present this is an internal library with an unstable API. Where necessary the functionality should be re-exported, for example like it is in the Rust-SDK.

This library should always be possible to compile for android, iOS, Wasm, and x86 code. Some parts may be feature gated to work around platform specific limitations though.

This library also exports other core crypto dependencies so that consumers may simplify their dependencies. Users are intended to get the re-exported dependencies through the library, instead of separately.
