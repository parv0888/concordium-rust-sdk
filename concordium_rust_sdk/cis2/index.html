Module concordium_rust_sdk::cis2[−][src]

Expand description

Types and functions for working with CIS-2 smart contracts. This module contains types and functions for interacting with smart contracts following the CIS-2 specification.

The type Cis2Contract act as a wrapper around the Client and a contract address providing functions for querying and making transactions to smart contract.
