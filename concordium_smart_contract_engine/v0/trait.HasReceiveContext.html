Expand description

Types which can act as receive contexts.

Used to enable partial JSON contexts when simulating contracts with cargo-concordium.

We have two implementations:

  • ReceiveContext, which is used on-chain and always returns Ok(..).
  • ReceiveContextOpt, which is used during simulation with cargo-concordium and returns Ok(..) for fields supplied in a JSON context, and Err(..) otherwise.
