Expand description

This function verifies that the signature inside the AccountOwnershipProof is a signature of the given message. The arguments are

  • keys - the verification keys
  • threshold - the signature threshold
  • proof_acc_sk - the AccountOwnershipProof containing the signature to be verified
  • msg - the message
