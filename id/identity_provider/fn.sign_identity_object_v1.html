Expand description

Sign the given pre-identity-object to produce a version 1 identity object. The inputs are

  • pre_id_obj - The version 1 pre-identity object
  • ip_info - Information about the identity provider, including its public keys
  • alist - the list of attributes to be signed
  • ip_secret_key - the signing key of the identity provider
