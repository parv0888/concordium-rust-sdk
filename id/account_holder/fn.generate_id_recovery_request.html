) -> Option<IdRecoveryRequest<C>>
Expand description

Generate a ID recovery request, proving of knowledge of idCredSec. The arguments are

  • ip_info - Identity provider information containing their ID and verification key that goes in to the protocol context.
  • context - Global Context containing g such that idCredPub = g^idCredSec. Also goes into the protocol context.
  • id_cred_sec - The secret value idCredSec that only the account holder knows.
  • timestamp - seconds since the unix epoch. Goes into the protocol context.
